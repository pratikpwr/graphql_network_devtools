# Claude Code Configuration

## Project Overview
GraphQL Network DevTools - A developer tool for monitoring and analyzing GraphQL network requests.

## Development Guidelines

### Branch Policy
- Develop on the designated feature branches specified in your session
- Commit changes with clear, descriptive messages
- Push to the branch when work is complete

### Code Style
- Follow existing code patterns and conventions in the repository
- Keep changes focused and minimal - avoid unnecessary refactoring
- Write clear commit messages that explain the "why" not just the "what"

### Testing
- Run tests before committing changes
- Ensure all existing tests pass with your modifications

### Git Workflow
- Use `git push -u origin <branch-name>` for branch pushes
- Create descriptive commit messages
- Do not force push to shared branches

## Useful Commands
```bash
npm test           # Run tests
npm run build      # Build the project
npm run dev        # Start development server
```
